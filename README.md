# wercker-dev

Wercker devcenter built with [gulp][gulp] and [metalsmith][metalsmith].

## Installation
```sh
git clone https://github.com/wercker/dev
```

Install node:
```sh
brew install nvm
nvm install 0.10
nvm alias default 0.10
npm install -g npm@2
```

## Building
Build:
```bash
make
```

Watch:
```sh
make watch
```
