# wercker-docs

Wercker docs built with [gulp][gulp] and [metalsmith][metalsmith].

## Installation
With [node][node] and [npm@2][npm] installed:
```sh
git clone https://github.com/wercker/dev
```

## Building
Build:
```bash
make
```

## Live reload while working

In one terminal run:
```sh
make watch
```

and in another run:

```sh
make open
```
